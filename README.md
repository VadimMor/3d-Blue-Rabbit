# 3d-Blue-Rabbit
🌐 Full-stack веб-сайт для студии 3D-печати  Разрабатываю современное полнофункциональное приложение с нуля.  Frontend: каталог моделей, калькулятор стоимости печати, оформление заказа, личный кабинет.  Backend: управление заказами, админ-панель, интеграции с принтерами и платежами.  Всё пишу сам - и клиентскую часть, и серверную.
